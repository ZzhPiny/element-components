# Element-Components
| 组件名称      | 描述            |
| ------------- | --------------- |
| dynamic-table | 列表      |
| dynamic-form  | 表单      |
| descriptions  | 详情      |


# Install
```
yarn add @ks/element-components
```

# Usage
查看文档:
```sh
npm run docs
```

文档更新:
```sh
npm run docs:gen
```

文档预览:
```sh
npm run docs:preview
```
